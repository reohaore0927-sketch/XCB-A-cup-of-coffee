# ☕ XCB-一杯のコーヒー

日本円 (JPY) を入力するだけで、リアルタイムレートで Core Blockchain (XCB) の枚数を自動計算し、CorePassアプリで読み取れる決済用QRコードを即座に生成するWebPOSアプリです。

少額決済（マイクロペイメント）に特化しており、操作ミスによる高額請求を防ぐ安全設計になっています。

📱 **レジアプリを開く:** https://reohaore0927-sketch.github.io/XCB-A-cup-of-coffee/

## ✨ 主な機能

* **CorePass完全対応:** 公式ウォレットで確実に読み取れる `payto://xcb/` 形式のQRコードを生成します。
* **500円上限リミッター:** 誤入力による高額請求を防ぐため、一回の支払いを最大500円までに制限しています。
* **リアルタイムレート:** CoinGecko APIを使用し、ワンタップで最新の XCB/JPY レートを取得します。
* **設定自動保存:** 受取用ウォレットアドレスはブラウザに記憶されるため、毎回入力する必要はありません。

## 🚀 使い方

### 初回設定
1.  アプリを開き、「受取アドレス」欄にご自身のXCBウォレットアドレスを入力します。
2.  入力すると自動的に保存されます。

### 決済手順
1.  「↻ 更新」ボタンを押して、最新のレートを取得します。
2.  「請求金額 (最大500円)」を入力します。
3.  「QRコード作成」ボタンをタップします。
4.  表示されたQRコードをお客様の CorePass アプリでスキャンしてもらいます。

## ⚠️ 注意事項

* 本アプリはインターネット接続が必要です。
* APIの通信状況により、レート取得に数秒かかる場合があります。
